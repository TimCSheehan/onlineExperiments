Series of experiments run on Amazon's mechanical Turk. All code heavily modified from original scripts by Tim Brady. "AdaptExperiment" was largely written by UG research assistant Muru Zhang.


AdaptExperiment - an adapting stimulus flashes for 5s at the start of the experiment. Participants will then make a sequence of delayed continuous report trials, interspersed with top-up adaptation.

CD : Change detection task. Grating followed by line.

UMI: Unattended Memory Item. Experiments exploring the effects of graded exogenous attention cues. Two gratings with a probabilistic pre/retro cue}
